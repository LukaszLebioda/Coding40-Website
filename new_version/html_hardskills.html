Source code editing

Notepad++

name -v: Notepad++
in three words:: The beauty of simplicity
long story short: A simple, light, fast and reliable text editor every front-end developer get in touch with from the very beginning. It's free, has a great logo, a straightforward interface and classical look that can be adjusted to a certain degree. No fancy formatting options, no code completions, no snippets; just raw source code with some syntax colorizing.
opinion: Love it, although for more complex coding it may be not enough.
image:

Brackets

name -v: Brackets
in three words:: Webdesign simplified
long story short: A free, multiplatform text editor from Adobe. Fast, easy to handle and nice-looking, with many plugins to enhance both look and mechanics. A live preview functionality allows to observe changes made when developing websites automatically, real-time, without even saving.
opinion: For website development one of the best tools out there.
image:

Visual Studio Code

name -v: Visual Studio Code
in three words:: A machine for writing code
long story short: One of the most popular code editors with vast online community, lots of tutorials and lots of plugins that make coding easier and code itself look prettier. It's free, multiplatform, supports almost every coding language syntax and has in-built terminal to rwx files.
opinion: A must-have. I can't image developing JavaScript apps without it.

Front-end

HTML

name -v: HTML 5 (HyperText Markup Language)
in three words:: Every beginners' beginning
long story short: a language to describe the content of the page and the general behaviour of the page in different environments, to link some external files or CDN libraries and to pass some info to the search engines out there
opinion: Quite intuitive and straigthforward at the beginning; hovever the deeper into the forest You go, the more trees You see...; without CSS it seems rough and naked (see CSS).
code sample:a simple HTML template with head section for metadata and body section for actual content.

 
<!doctype html>
<html>
<head>
    Meta info goes here.
<title>HTML Template<title>
<meta name="description" content="html, template">
<meta name="keywords" content="html tutorial template">
<head>
<body>
    Content goes here.
<body>
<html>

CSS

name -v: CSS 3 (Cascading Style Sheets)
in three words:: Instant pleasure of design
long story short: a language to tell the browser how it should render the page's look: fonts, colours, images, backgrounds, icons and many more; a powerful tool that is firmly bound with html and can be supported by external libraries to create even more picturesque results
opinion: Nice and clean syntax that tends to get more and more complicated in the course of development; can be tricky to remember what lines of css represent what, so comments every now and then are strongly recommended.
code sample: a css syntax sample that sets the page background color blue, describes the font-look of a main header and a paragraph and sets the resolution of an image.

 
body {
  background-color: lightblue;
}

h1 {
  font-family: verdana;
  font-size: 20px;
  color: white;
  text-align: center;
}

img {
  width: 700px;
  height: 300px;
}

JavaScript

name -v: Java Script (ECMA Script 6)
in three words:: It's a kind of magic
long story short: A programming language that has become a core technology to develop modern www pages. It's syntax has the magnificent ability to manipulate DOM, which injects interactivity and life info traditional html-css structure. Many frameworks and runtimes use JavaScript nowadays, including Cypress (testing) and node.js (back-end programming).
opinion: Although it's syntax seems a little bit overloaded (in contrast to python), it provides the opportunity to dive deep into the programming world and the results can be truly satisfactory.
code sample: that tiny little piece of js below marks a certain “element” of the page and attaches it to one of the js typical events - a “ click” event that triggers a function; a solely purpose of that function's existence is to pop up a window with “Hello World!” message.

 
element.addEventListener("click", myFunction);

function myFunction() {
  alert ("Hello World!");
}

jQuery

name -v: jQuery
in three words: JavaScript simplified.
long story short: jQuery is a JavaScript library: a set of ready-to-go pieces of code (a set of functions to invoke actually) we can use when we want to interact with a DOM element - to change it, hide it, expose it, animate it etc. In other words it's main purpose is to make the manipulation of DOM elements easier according to the rule: "Write less do more".
opinion: You really can achieve more with less code written. The syntax is straightforward and simple to learn, the mechanics is obvious and easy to apply. Another tool to have in a webdev-garage.
code sample: The code below makes sure the page is fully loaded. Then it waits for an element (in our example: "p", a paragraph) to be clicked. When this happens the clicked element dissapears within 2000 miliseconds.

 
$(document).ready(function(){
  $("p").click(function(){
    $(this).fadeOut(2000);
  });
});

Bootstrap

name -v: Boostrap 5
in three words: Freedom for responsiveness
long story short: A framework to build responsive websites. Basicly it's an extensive css file that contains responsive grid mechanism and bunch of pre-definded styling options designed to replace or co-operate with css syntax. In comes with many in-build elements to be used: forms, pop up windows, navigation bars, progress bars and many many more.
opinion: It is possible to build a fully responsive website with css only and have unlimited freedom of choice how the design should be. It is easier to use bootstrap instead.
code sample: Within the main div classed "container" (of blue back-ground and with white font text) there is another one classed "row". Everything that is placed within the latter behaves responsively depending on the viewport sizes, called breaking points in bootstrap (sm, md, lg, xl). 

 
<div class="container bg-primary text-white">
  <div class="row">
    <div class="col-sm-12">
    </div>
    <div class="col-md-6">
    </div>>
    <div class="col-md-6 col-lg-4">
    </div>
  </div>
</div>
