Explanation: This is only a game. Please do not make any decisions based upon provided results! The codewriter disclaims any responsibility for what may or may not happen to You if You will!
Source code: The code I wrote reflects ruthlessly the level of my initiation. It's a bit messy and probably too repetitive. But on the other hand it gave me so much fun having made the program work and hope that I would do better! Check the source code if You will at my GitHub repository or install npm package directly into Your project with: npm instal legalitychecker.
